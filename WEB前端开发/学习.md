- html决定了网页的结构

- css是写在html文件里面的，决定了样式

- js决定了网页的交互

- vs code 感叹号可以召唤html的结构

- `<br/>`代表回车，意思是break line

- 在html中，不管中间有多少回车和换行都被解析为**一个**空格

- html的转义字符

  - `&ensp;`代表空格

- `<!--  -->`

- ```html
  <html lang="zh-CN">
      <head>
          配置
          <meta charset="UTF-8">
          <title>庞梦微</title>
      </head>
  
      内容
      <body>
  
          
          hello  world \n     
      
  
          &lt;
        
          你  &ensp;	&ensp;	&ensp;	&ensp;	&ensp;	    好     世     界
      </body>
  </html>
  
  ```

  ```html
      <ol>
