# 小课堂系列：全领域选题大百科【选题推荐】

各位同学，经过前面几节课的学习，我们已经把GameJam史、MDA理论、4X准则和游戏类型分类收入囊中。但我知道，你们的好奇心远不止于此！游戏的世界太大了，美术、音乐、叙事、哲学、技术、宣发……每一个角落都藏着值得深挖的宝藏。

今天，我们就来一次“全领域扫荡”，看看还有哪些“小课堂”可以安排上！**（多图预警，脑洞预警，作业预警）**

---

## 🎨 美术篇：游戏是第九艺术，那美术就是艺术的脸面

### 1. 像素艺术简史：为什么马赛克永远不过时？

- **内容**：从70年代街机的点阵，到如今独立游戏的复古风潮。像素画为什么有魔力？色数限制、抖动技术、像素动画的黄金法则。
    
- **代表**：《铲子骑士》《星露谷物语》《八方旅人》（HD-2D也算！）
    
- **冷知识**：早期像素画师为了省颜色，会把两个颜色交织在一起产生第三种颜色的视觉效果，这叫“抖动”（dithering）。
    

### 2. 概念设计：游戏角色是怎么“生”出来的？

- **内容**：从草图到3D模型，角色设计经历了什么？剪影测试、色彩情绪、文化元素融合。那些废案（比如《守望先锋》早期设计）有多离谱？
    
- **代表**：《最终幻想》的天野喜孝、《光环》的Spartan设计、《塞尔达》的幻想风格。
    
- **小作业**：画一个“洋葱骑士”的剪影，让室友猜是哪个游戏角色。
    

### 3. UI/UX设计：按钮放在哪，玩家才不骂娘？

- **内容**：Fitts定律、希克定律、格式塔原理……这些心理学知识怎么用在游戏界面里？为什么《生化危机4》的背包界面一直被模仿？为什么手游的充值按钮总是那么显眼？
    
- **案例**：《死亡空间》的“全息投影”UI、《暗黑破坏神》的技能栏进化史。
    

### 4.游戏色彩心理学：为什么敌人都是红色的？

- **内容**：为什么反派总是穿红黑？为什么治愈是绿色？色彩如何影响情绪和游戏体验？《镜之边缘》的纯白世界、《空洞骑士》的苍白配色。
    
- **拓展**：色盲模式的设计，不只是为了政治正确。

## 🎵 音乐篇：没有BGM，游戏就只剩操作了

### 1. 游戏音乐进化论：从8bit到交响乐

- **内容**：红白机的三角波、SFC的音源芯片、PS1的CD音轨、到现在的动态交响乐。音乐怎么从“滴滴答答”变成“史诗级BGM”？
    
- **里程碑**：《超级马里奥》的地下音乐、《最终幻想7》的一首《片翼天使》、《尼尔》的《遊園施設》。
    
- **冷知识**：早期游戏音乐家都是程序员兼职的，后来才有了植松伸夫、近藤浩治这样的专业作曲。
    

### 2. 音效设计：一扇门怎么发出“高级感”？

- **内容**：拟音艺术——用芹菜折断的声音模拟骨头碎裂，用椰子壳模拟马蹄声。《最后的生还者》里蘑菇头的声音是怎么做的？
    
- **案例**：《战地》的“战争回响”系统、《寂静岭》的工业噪音。
    
- **小实验**：用身边的东西录一段“脚步声”，看看像不像游戏里的。
    

### 3. 动态音乐系统：BGM会看眼色行事？

- **内容**：地平线系统（layered music）、横向重编曲（horizontal resequencing）。为什么进入战斗音乐会变？为什么危险时鼓点会加速？
    
- **代表**：《超级马里奥银河》的“引力音乐”、《节奏地牢》的“随节拍移动”。
    

### 4. 音乐游戏简史：从《啪啦啪啦啪》到《节奏光剑》

- **内容**：音游的起源、发展、没落与复兴。为什么音游玩家都是“触手怪”？
    
- **代表**：《DDR》《太鼓达人》《osu!》《Cytus》《节奏光剑》。
    

---

## 📖 叙事学篇：游戏不只是“玩”，更是“读”

### 1. 游戏叙事学导论：故事是怎么被“玩”出来的？

- **内容**：游戏叙事和小说、电影的根本区别：互动性。线性叙事、分支叙事、环境叙事、碎片化叙事，每种都有什么魔法？
    
- **代表**：《最后生还者》（线性）、《巫师2》（分支）、《黑暗之魂》（碎片）、《生化奇兵》（环境）。
    
- **经典理论**：ludonarrative dissonance（玩法与叙事的冲突）——为什么《神秘海域》里德雷克杀人如麻却依然是个快乐冒险家？
    

### 2. 环境叙事：不说话，让场景自己讲故事

- **内容**：通过场景细节传递故事——墙上的弹孔、地上的日记、尸体的姿势。玩家自己拼凑出来的故事，最有代入感。
    
- **代表**：《生化危机》的文档、《耻辱》的录音、《艾迪芬奇的记忆》的幻想场景。
    

### 3. 分支叙事：选择真的有意义吗？

- **内容**：蝴蝶效应 vs 伪选择。为什么《底特律：变人》的选择让你纠结？为什么《奇异人生》只能二选一？如何让玩家觉得“我的选择重要”？
    
- **技术**：状态机、故事图、动态对话系统。
    

### 4. 情感设计：怎么让玩家哭成狗？

- **内容**：游戏如何调动情绪？音乐、画面、玩法、叙事的综合运用。《去月球》的像素催泪、《传说之下》的“不杀一人”结局、《FF7》爱丽丝之死为什么成为经典？
    
- **心理学**：移情、依恋、损失厌恶。

## 🎮 游戏学篇：游戏之所以是游戏的“为什么”

### 1. 心流理论：为什么一玩就停不下来？

- **内容**：心理学家米哈里提出的心流状态——当挑战和技能完美匹配时，人会进入“忘我”状态。游戏怎么设计难度曲线？怎么防止无聊和焦虑？
    
- **案例**：《植物大战僵尸》的渐进式难度、《黑暗之魂》的“受苦但能过”。
    

### 2. 玩家类型学：你是“成就党”还是“探索党”？

- **内容**：巴图玩家分类法（成就者、探索者、社交者、杀手）及其扩展。为什么有人喜欢收集全成就，有人只想逛地图？
    
- **应用**：游戏设计如何满足不同玩家？《星露谷物语》的多种玩法路径。
    

### 3. 游戏平衡的艺术：为什么“毒瘤”总被削？

- **内容**：平衡性的本质是什么？对称平衡 vs 非对称平衡。为什么剪刀石头布是终极平衡？为什么《星际争霸》三族能平衡？
    
- **方法论**：数据驱动、玩家反馈、职业比赛调整。
    

### 4. 游戏文化研究：玩家社群是怎么形成的？

- **内容**：从论坛到Discord，从速通到MOD，玩家社群如何影响游戏？《我的世界》的MOD生态、《上古卷轴》的“实验室”、《Among Us》的梗文化。
    
- **社会学**：共同体验、集体记忆、粉丝创作。

## 🤔 游戏哲学篇：玩游戏到底在玩什么？

### 1. 游戏的本质：什么是“游戏”？

- **内容**：从赫伊津哈的“魔环”理论，到麦格尼格尔的“游戏四大特征”。游戏和非游戏的界限在哪？《模拟人生》是游戏还是玩具？
    
- **哲学家**：维特根斯坦说“游戏无法定义”——我们用家族相似性来理解。
    

### 2. 存在主义与游戏：自由意志的幻觉

- **内容**：萨特的“存在先于本质”——在游戏里，玩家的选择真的自由吗？还是说只是开发者预设的路径？《史丹利的寓言》怎么讽刺选择权的？
    
- **思考**：如果游戏是全知全能的开发者创造的，玩家还有自由吗？
    

### 3. 游戏伦理学：在游戏里杀人算不算道德问题？

- **内容**：虚拟暴力会不会导致现实暴力？游戏中的道德选择（比如《辐射3》的核弹镇）反映了什么？我们为什么对NPC产生道德感，对敌人却麻木？
    
- **案例**：《传说之下》的“不杀”机制、《GTA》的道德困境。
    

### 4. 游戏与后现代：解构、戏仿、元游戏

- **内容**：游戏如何打破第四面墙？《死侍》的自我吐槽、《尼尔》的多周目、《史丹利的寓言》的旁白。游戏是对现实的模拟，还是对模拟的模拟？

## ⚙️ 技术篇：游戏背后的代码魔法

### 1. 游戏引擎简史：从自己造轮子到Unity/UE称霸

- **内容**：90年代每个公司都有自己的引擎（id Tech、虚幻1），到现在Unity和UE二分天下。引擎到底提供了什么？为什么独立开发者都用Unity？
    
- **里程碑**：《毁灭战士》的3D引擎、《虚幻》的光照技术、《塞尔达旷野之息》的物理引擎。
    

### 2. 渲染技术入门：光追到底牛在哪？

- **内容**：顶点、像素、着色器、光照模型。从固定管线到可编程管线，从光栅化到光线追踪。为什么《赛博朋克2077》的光追那么吃显卡？
    
- **概念**：法线贴图、环境光遮蔽、全局光照。
    

### 3. 物理模拟：为什么《GTA》的车会翻？

- **内容**：刚体物理、软体物理、布娃娃系统、破坏效果。Havok和PhysX是什么？《愤怒的小鸟》的物理是怎么实现的？
    
- **案例**：《传送门》的精妙物理、《战神》的混沌之刃链条模拟。
    

### 4. 网络同步：延迟这么高，我怎么还能打死人？

- **内容**：CS的“延迟补偿”、格斗游戏的“回滚代码”、RTS的“锁步同步”。为什么《黑暗之魂》的联机这么怪？
    
- **难题**：防外挂、防作弊、状态同步 vs 帧同步。
    

### 5. AI设计：NPC为什么有时候很蠢？

- **内容**：有限状态机、行为树、导航网格、寻路算法。为什么《最后生还者》的艾莉这么聪明？为什么《模拟人生》的小人会做蠢事？
    
- **经典**：《F.E.A.R》的战术AI、《求生之路》的“导演系统”。

## 📣 游戏宣发篇：好酒也怕巷子深

### 1. 游戏营销简史：从杂志广告到TikTok病毒

- **内容**：90年代靠游戏杂志封面，00年代靠电视广告，10年代靠KOL试玩，20年代靠短视频。游戏是怎么“卖”出去的？
    
- **案例**：《无人深空》的过度宣传、《赛博朋克2077》的基努宣传、《Among Us》的三年后爆红。
    

### 2. 独立游戏怎么推广？穷鬼的生存指南

- **内容**：社交媒体、游戏展会（PAX、indieCade）、众筹、媒体通稿、折扣策略。没有预算怎么办？
    
- **案例**：《星露谷物语》的四年开发+Reddit连载、《传说之下》的口碑奇迹。
    

### 3. 社区运营：怎么把玩家变成“自来水”？

- **内容**：Discord管理、粉丝创作激励、沟通艺术。为什么《无人深空》能逆转口碑？为什么有些官方下场被喷？
    
- **雷区**：画大饼、怼玩家、过度商业化。

### 4. 游戏品牌建设：从IP到粉丝经济

- **内容**：如何把一个游戏做成IP？周边、影视改编、联动。《巫师》靠游戏带火原著，反过来又带动游戏销量。
    
- **案例**：《精灵宝可梦》的帝国、《原神》的全球布局。
