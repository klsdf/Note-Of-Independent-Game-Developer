# 目的

1. 游戏开发的门槛不断降低，让全栈开发成为可能。
    
2. 很多从业者的技术栈不全面，而且不知道怎么样补全，浪费大量时间探索绕弯路。
    
    1. 例如浪费大量时间学习数据结构，学习厚涂，学习复调音乐。
        
3. 本课程可以让观众拥有最最基本的独立游戏开发能力，帮你用最短的时间速成全栈开发者。
    

**（认知）**

4. 注意，全栈意味着全会，但是不意味着全精通。你无法通过本课程成为一个职业程序员或者作曲家。
    
5. 大部分的游戏制作要远低于我们的想象。制作游戏最难的其实是广度，而不是深度。
    
6. 本课程既可以当作大家查漏补缺的字典，也可以作为快速入门的指南。

# 目标预期

## 知识预期

两年时间内掌握独立游戏开发的基本知识，下面是基本的预期

- **拥有较系统的独立策划的能力**
    
    - 有基本的游戏常识
        
    - 能掌握基本的游戏史
        
    - 对各种游戏的设计难点和设计要点有了解
        
    - 能明白自己到底要做什么样的游戏
        
    - 能明白如何开始设计游戏
        
    - 明白如何迭代游戏
        
    - 明白怎么样去分析和拆解游戏
        

  

- **拥有基本的绘制美术素材的能力，和美术的审美**
    
    - 能绘制基本的二分
        
    - 能大概判断某个游戏的画风是什么
        
    - 能看出来一张画哪里好，哪里不好
        
- **拥有最基本的编剧能力**
    
    - 掌握基本的故事结构和叙事知识
    - 能完整把一个场景，写成完整的剧本。


- **拥有基本的建模和贴图能力**
    
    - 能用blender完成最简单的家具建模
        
    - 能制作基本的贴图
        
- **拥有基本的作曲和音效能力，和音乐审美**
    
    - 能听出来某个游戏的音乐风格是什么
        
    - 能制作简单的bgm和se
        
    - 会简单使用音频中间件
        
- **拥有基本的游戏客户端开发能力**
    
    - c#基础（写的不够详细）
        
    - unity开发
        
    - 基本的设计模式
        
- **拥有基本渲染的知识**
    
    - 了解基本的渲染管线知识
        
    - 能看懂shader（不用会写）
        
    - 能使用shadergraph完成常见的特效和后处理效果
        
- **拥有基本的游戏运营能力**
    
    - 制作宣传视频
        
    - 海报制作
    - 知道有哪些宣发平台，这些平台有哪些偏好。
        
    - 知道如何上架steam
        
    - 知道如何上架itch
        
- **拥有游戏行业的较全面认知**
    
    - （如何找到信息的来源）
        
    - 国内有哪些游戏公司
        
    - 这些公司的游戏有哪些
        
    - 这些游戏的运营状况如何，受众群体是谁
        
    - 全世界有哪些知名的发行商，他们偏好什么样的游戏
        
    - 国内和国际有哪些知名游戏展
        
    - 国内和国际有哪些知名gamejam
        

  

- **熟练掌握各种游戏小工具以及如何熟练使用AI**
    
    - 能熟练使用ai绘图和ai作曲，来帮自己快速完成游戏原型
        
    - 数量掌握comfyui的使用
        
    - 浏览器收藏夹中有不少于10个素材网站和资源网站
        

  

## 阶段预期

1. 半年内，可以参加gamejam并担任一个职位
    
2. 一年内，可以solo gamejam
    
3. 两年内，可以制作一个比较完整独立游戏demo，并上架并上架steam页面或者itch页面（上商店页面不代表要发售或者赚钱）
