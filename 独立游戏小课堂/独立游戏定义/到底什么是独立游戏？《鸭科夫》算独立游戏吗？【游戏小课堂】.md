
国内外每天上架的独立游戏多如牛毛，虽然很多人都标榜自己的游戏是“独立游戏”，但像《光与影：33号远征队》这样的大体量作品，依然引发了不小的争议。

那么，到底什么样的游戏才算独立游戏？独立游戏又该如何定义？
其实这类定义问题并没有统一答案。（要不然就不会有争议了）

所以，今天我们就来看看“独立游戏”的主流定义有哪些吧~

#  Jesper Juul的定义

在讲独立游戏之前，我们先来看看远方的独立电影吧！
独立游戏来源于英文 "Indie Game" 。而 "Indie Game"则是 "Independent Game" 的缩写，而这个概念最初源于独立音乐和独立电影。
总的来说就是，一帮子搞艺术的人，不满于被邪恶的资本主义异化！因此完全和各种商业音乐、电影唱反调！主打一个叛逆。之后，这种精神很快传染了游戏行业，也诞生出了独立游戏。

2005年，Geoff King 在其著作《American Independent Cinema》中，从3个维度确定了独立电影的基本定义：
- 产业 / 资金维度（Industrial location）：与好莱坞主流工业体系的分离
- 形式 / 美学维度（Formal/aesthetic strategies）：独特的叙事与视觉风格
- 社会 / 文化 / 意识形态维度（Social/cultural/political landscape）：对主流价值观的反思与批判

而游戏学者Jesper Juul则在其基础上，撰写了著作 **《Hand-made Pixels: Independent Video Games and the Quest for Authenticity》（手作像素：独立游戏及对真实性的探索）**，并定义了独立游戏的3个特征：

1. 资金独立，游戏开发者不能受资本的裹挟，必须自由地进行更具个性化的创作。
2. 美学独立，必须采用不同于主流游戏的作品风格和设计原则。
3. 文化独立，游戏应该承载着文化的、政治的和道德的价值，从而让世界变得更美好。
![[image-3.png]]
Jesper Juul


这个定义可以说是极其学院派了，但是也不是铁板一块。

比如说，由Nosotek开发的知名游戏《平壤赛车》（Pyongyang Racer）就是一个例子。

《平壤赛车》从定义来看完全符合独立游戏的定义：
- **资金独立**：由旅行社委托开发，无大型发行商 / 资本强制投资。最终由Nosotek 主导开发，没有任何资本的裹挟，不为盈利而生
- **美学独立**：独特的主体审美，可以欣赏独具一格的平壤美景。完全区别于主流 3A 赛车游戏！
- **文化独立**：承载朝鲜城市文化与旅游宣传价值，具有明确文化表达

但是要说《平壤赛车》是一款优秀的独立游戏，又总感觉怪怪的。。
所以说Jesper Juul的定义是无法让所有人满意的。

# Wiki的定义


另一方面，Jesper Juul的定义未免过于苛刻，对于很多独立游戏制作人来说，资金独立是极其困难的。就好比独立游戏届的”“败犬宠儿“——《Fez》也曾获得过加拿大政府贷款和魁北克开发商 Trapdoor 的财务支持。但是它是公认的独立游戏。

![[独立游戏小课堂/独立游戏定义/asserts/image-1.png]]


因此wiki百科上的定义也更加柔和一些：

> 独立游戏（英语：independent game 或 indie game）一般指游戏开发者未接受商业投资，独立负担开发成本的游戏。也可指具有独立精神，在玩法、风格等方面不追随主流市场，富有创意的游戏。

也就是说，你要么资金独立，要么有独立精神。都可以。

当然，我们注意到wiki的这个词条来源其实是《中国大百科全书》的独立游戏定义
https://www.zgbk.com/ecph/words?SiteID=1&ID=442345&Type=bkzyb&SubID=119459

而这个定义的来源并不可考，所以这个定义可能并不是游戏业内的普遍观点。大家可以辩证看待。


# 普遍定义


接下来就是玩家之间的约定俗成的定义了，并没有明确的出处，只能说是我汇集了多个文章的观点整合的定义。世界上本没有定义，我寻思是这样，你也寻思这样，慢慢就有了定义。

1. **具有独立精神**：在游戏机制或者美术、叙事等任何一个角度有着疯狂探索，完全和主流的商业游戏背道而驰，而不担心因此赚不到钱。
2. **工作室人数少于10人**：或者说参与游戏开发的人数少于10人。不能像《33号远征队》一样找一堆外包。
3. **财务独立**：不依赖外部资本、自筹资金
4. **创作独立**：就算去拉投资了，创作也必须保持独立性，不能被发行商、投资人要挟，破坏开发商的自主性
