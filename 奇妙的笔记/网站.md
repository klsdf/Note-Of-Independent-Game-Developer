
# Processing

Processing 是一个灵活的软件草图本，也是一门学习如何编程的语言。自 2001 年以来，Processing 在视觉艺术领域推广软件素养，在技术领域推广视觉素养。有成千上万的学者、艺术家、设计师、研究人员和爱好者使用 Processing 进行学习和原型设计。

https://processing.org/
