


# 站内搜索分析（SSA/Site Search Analytics）

用户在**网站内部**搜索了什么

## 齐普夫定律（Zipf's Law）


 排名第 n 的词或对象的频率 ≈ 排名第一的频率 ÷ n



|排名|词语|出现频率|
|---|---|---|
|1|“的”|1000次|
|2|“是”|500次|
|3|“在”|333次|
|4|“我”|250次|



齐普夫定律的分布规律可以精确判断应该优化哪 些搜索词条,从而产生最好的效果。
