#### World Space —— 世界模式

| 属性                      | 功能                                                         |
| ------------------------- | ------------------------------------------------------------ |
| Event Camera              | 响应事件的相机                                               |
| Sorting Layer             | 画布的深度,指定了相机的渲染顺序                              |
| Order in Layer            | 值越大，该UI越显示在前面                                     |
| Addtional Shader Channels | 附加着色通道，决定Shader可以读取哪些相关数据，比如 法线、 切线 等数据 |
