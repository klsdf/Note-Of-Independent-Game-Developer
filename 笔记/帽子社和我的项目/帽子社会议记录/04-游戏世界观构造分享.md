# 游戏世界观构造分享会

# 规则

1. 不深究设计理论，重在分享和经验交流
2. 讲解世界观的内容，以及自己创造世界观的方法
3. 最好加上自己的游戏演示，或者插图说明
4. 所有世界观没有好坏之分

# 流程

1. 会议的规则
2. 主讲人进行分享示范
3. 其余人自愿分享
4. 一个人分享结束后，其余人可以进行提问
5. 投票选择自己喜欢的世界观。（无好坏之分，只是衡量人气）
6. 会议笔记总结，视频发布
