# 快捷键

| 快捷键                    | 意思               |
| ------------------------- | ------------------ |
| <kbd>A</kbd>              | 全选               |
| <kbd>shift + A</kbd>      | 在游标处创建物体   |
| <kbd>X</kbd>              | 删除               |
| <kbd>shift + C</kbd>      | 游标归零           |
| <kbd>鼠标中键+x,y,z</kbd> | 沿着xyz轴移动      |
| <kbd>G</kbd>              | 平移物体           |
| <kbd>R</kbd>              | 旋转物体           |
| <kbd>S</kbd>              | 缩放物体           |
| <kbd>TAB</kbd>            | 进入物体的编辑模式 |
| 编辑模式下123             | 切换点线面         |
| ctrl+R                    | 切割物体           |
|                           |                    |
| 小键盘1                   | 主视图             |
| ctrl+小键盘1              | 反主视图           |
| i                         | 内插面             |
