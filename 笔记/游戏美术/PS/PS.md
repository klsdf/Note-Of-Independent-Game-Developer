# 快捷键

| 组合             | 功能         |
| ---------------- | ------------ |
| shift + ctrl + N | 新建图层     |
| V                | 移动整个图层 |
| shift            | 画标准图形   |
| M                | 矩形选框工具 |
| L                | 套索工具     |
| ctrl + D         | 取消选框     |
| B                | 画笔         |
| [                | 缩小画笔大小 |
| ]                | 放大画笔大小 |
| 画笔模式下+alt   | 取色         |
| E                | 橡皮         |
| alt+delete       | 填充前景     |
